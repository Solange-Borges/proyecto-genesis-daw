# Proyecto Génesis

Web corporativa desarrollada como parte del proyecto intermodular de DAW.

## Descripción
Proyecto Génesis es una plataforma de formación bíblica para iglesias, familias y educadores.

## Páginas
- Inicio
- Nosotros
- Recursos
- Tribu
- Contacto

## Tecnologías
- HTML
- CSS
- JavaScript

## Cómo usar
Abrir index.html en el navegador
