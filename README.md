# 🌱 Proyecto Génesis – Portal Web Educativo Bíblico

## 📌 Descripción del proyecto

Proyecto Génesis es un portal web educativo diseñado para ayudar a iglesias, familias y maestros a enseñar la Biblia a niños de forma clara, estructurada y con propósito.

El objetivo principal es formar identidad, convicción y propósito en la infancia a través de recursos bíblicos adaptados por edades.

Este proyecto ha sido desarrollado como parte del **Proyecto Intermodular de 1º DAW**, integrando conocimientos de HTML, CSS, JavaScript y organización de proyectos.

---

## 🎯 Problema que resuelve

Muchos padres, maestros y líderes infantiles desean enseñar la Biblia, pero no siempre cuentan con herramientas claras, estructuradas y prácticas.

Proyecto Génesis ofrece una solución mediante:

* Recursos organizados por edades
* Material visual y didáctico
* Estructura clara para enseñanza bíblica

---

## 👥 Público objetivo

* Iglesias (ministerio infantil)
* Familias
* Maestros y educadores

---

## 🧩 Estructura del sitio web

El portal está compuesto por varias páginas principales:

* **Inicio (index.html)** → Presentación del proyecto
* **El Proyecto (about.html)** → Visión, misión y fundamentos
* **La Tribu (tribu.html)** → Modelo educativo y personajes
* **Recursos (recursos.html)** → Materiales y filtros por edad
* **Contacto (contacto.html)** → Formulario de comunicación

---

## 💻 Tecnologías utilizadas

* HTML5 (estructura semántica)
* CSS3 (diseño modular y responsive)
* JavaScript (interactividad básica)
* Git y GitHub (control de versiones)

---

## 🎨 Características principales

* Diseño responsive adaptable a diferentes dispositivos
* Navegación clara e intuitiva
* Menú dinámico reutilizable (header y footer cargados con JavaScript)
* Animaciones con IntersectionObserver
* Sistema de filtrado de recursos por edad
* Organización modular de estilos CSS

## ▶️ Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio
2. Abrir la carpeta del proyecto
