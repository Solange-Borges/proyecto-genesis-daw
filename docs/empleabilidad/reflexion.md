# Reflexión personal

En este proyecto he aprendido a organizar un proyecto web completo, trabajando la estructura HTML, el diseño con CSS y la interactividad con JavaScript.

Lo que mejor se me ha dado ha sido la parte visual y la organización del proyecto.

Lo que más me ha costado ha sido entender la estructura global del proyecto y la integración de todos los módulos.

En el futuro mejoraría la conexión con base de datos y añadiría más funcionalidades.
