
# Informe de Sistemas – Proyecto Génesis

## Tipo de sistema
Aplicación web ejecutada en un PC de usuario mediante navegador.

## Sistema operativo
Compatible con Windows, Linux o MacOS.

## Requisitos hardware
- CPU: básica (cualquier procesador actual)
- RAM: 4GB mínimo
- Almacenamiento: mínimo espacio para archivos del proyecto

## Instalación
1. Descargar el proyecto
2. Abrir la carpeta
3. Ejecutar el archivo index.html en un navegador

## Uso
El usuario navega por las distintas secciones de la web mediante el menú principal.

## Mantenimiento
- Actualizar contenido web
- Revisar enlaces y recursos
- Mejorar funcionalidades

## Seguridad básica
- No se manejan datos sensibles
- Uso de formularios externos (formsubmit)
